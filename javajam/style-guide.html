Logo

Java Jam

The Java Jam Logo Can be displayed in any of the colors from our color pallette. As well it should range from 1.5em to a a max of 2.5em for web. For print materials it should not exceed 1" high by 1.75" wide

Color Scheme

#4c1b1e Burnt Coffee

#fff Whipped White

#000 Grounds Black

TypeFaces

The Main font for the logos and headings is Leckerli One Also the supporting fonts for body text is Century Gothic or a different Sans-Serif Font

Images

All images within the site should be real images no illustrations are to be used.
